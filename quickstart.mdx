---
title: Quickstart
description: Get AgentRein protecting your agent in under 5 minutes.
---

## 1. Install the SDK
```bash
npm install agentrein
```

## 2. Get Your API Key

Log in to [Dashboard](https://www.agentrein.com/sessions), go to **Settings**, and copy your API key.

## 3. Initialize the Client
```ts
import { AgentRein } from 'agentrein';

const agentrein = new AgentRein({
  apiKey: process.env.AGENTREIN_API_KEY,
});
```

## 4. Create a Session
```ts
const session = await agentrein.newSession({
  agentId: 'my-agent',
  intent: 'Cancel the subscription for customer cus_xxx and notify them on Slack',
});
```

## 5. Wrap Your Clients
```ts
import { AgentRein } from 'agentrein';
import { Octokit } from '@octokit/rest';
import { WebClient } from '@slack/web-api';

const octokit = new Octokit({ auth: process.env.GITHUB_TOKEN });
const slackClient = new WebClient(process.env.SLACK_TOKEN);

const agentOctokit = agentrein.wrap(octokit, session, { connector: 'github' });
const agentSlack = agentrein.wrap(slackClient, session, { connector: 'slack' });

// Use exactly as before
const issue = await agentOctokit.issues.create({
  owner: 'my-org',
  repo: 'my-repo',
  title: 'Onboarding task',
});

const msg = await agentSlack.chat.postMessage({
  channel: '#onboarding',
  text: `Issue created: ${issue.data.html_url}`,
});
```

If any call throws, AgentRein automatically rolls back all previous actions in the session.

## 6. Complete the Session
```ts
await agentrein.completeSession(session);
```
