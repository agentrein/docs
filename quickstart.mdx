---
title: Quick Start
description: Get AgentRein protecting your agent in under 5 minutes.
---

## 1. Install the SDK
```bash
npm install @agentrein/sdk
```

## 2. Get Your API Key

Log in to [app.agentrein.com](https://app.agentrein.com), go to **Settings**, and copy your API key.

## 3. Initialize the Client
```ts
import { AgentRein } from '@agentrein/sdk';
import { Octokit } from '@octokit/rest'; // Developer's own standard SDK

const agentrein = new AgentRein({ apiKey: 'YOUR_API_KEY' });
const octokit = new Octokit({ auth: 'GITHUB_TOKEN' });
```

## 4. Create a Session
```ts
const session = await agentrein.newSession({
  agentId: 'onboarding-agent',
  intent: 'Create a GitHub issue and notify the team on Slack',
});
```

## 5. Wrap Your Existing Clients

Bring your own standard SDKs. Wrap them once and use them forever.
```ts
// The Magic: Wrap the client once
const agentOctokit = agentrein.wrap(octokit, session, { connector: 'github' });
```

## 6. Use Them Exactly As Before

Use the SDK exactly as normal. AgentRein handles logging and rollbacks automatically!

```ts
await agentOctokit.issues.create({ owner: 'my-org', repo: 'my-repo', title: 'Onboarding task' });
```

If any action throws, AgentRein automatically reverses all previous actions. The session always ends in a consistent state.

## 7. Complete the Session
```ts
await agentrein.completeSession(session);
```
