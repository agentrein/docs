<Steps>
  <Step title="Install the SDK">
```bash
npm install agentrein
```
  </Step>

  <Step title="Get Your API Key">
Log in to [Dashboard](https://agentrein.com/dashboard), go to **Settings**, and copy your API key.
  </Step>

  <Step title="Initialize the Client">
```ts
import { AgentRein } from 'agentrein';
import { Octokit } from '@octokit/rest'; // Developer's own standard SDK

const agentrein = new AgentRein({ apiKey: 'YOUR_API_KEY' });
const octokit = new Octokit({ auth: 'GITHUB_TOKEN' });
```
  </Step>

  <Step title="Create a Session">
```ts
const session = await agentrein.newSession({
  agentId: 'onboarding-agent',
  intent: 'Create a GitHub issue and notify the team on Slack',
});
```
  </Step>

  <Step title="Wrap Your Existing Clients">
```ts
const agentOctokit = agentrein.wrap(octokit, session, { connector: 'github' });
const agentSlack   = agentrein.wrap(slackClient, session, { connector: 'slack' });

const issue = await agentOctokit.issues.create({
    owner: 'my-org',
    repo: 'my-repo',
    title: 'Onboarding task',
});

await agentSlack.chat.postMessage({
    channel: '#onboarding',
    text: `Issue created: ${issue.data.html_url}`,
});
```
  </Step>

  <Step title="Complete the Session">
```ts
await agentrein.completeSession(session);
```
  </Step>
</Steps>

## Plan Tiers & API Keys

AgentRein has four plan tiers: **FREE**, **PRO**, **TEAM**, and **ENTERPRISE**.

Your API key works across all plans, but certain features require a minimum plan:

| Feature | FREE | PRO | TEAM |
|---|---|---|---|
| Session Rollback | ✓ | ✓ | ✓ |
| API Key Management | ✓ | ✓ | ✓ |
| Connector Credentials | ✓ | ✓ | ✓ |
| Approval Gates (HITL) | ✗ | ✓ | ✓ |
| Intent Drift Detection | ✗ | ✗ | ✓ |

<Note>
If you call a PRO or TEAM feature on a FREE plan, the backend returns a `403` 
with a clear error message. Upgrade at [Dashboard → Settings → Billing](https://agentrein.com/dashboard/settings/billing).
</Note>

To get your API key: **Dashboard → API Keys → Create New Key**.
