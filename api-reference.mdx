## `wrap<T>`

Wrap a standard SDK client to enable automatic logging, interception, and rollbacks.

**Signature:**  
```ts
wrap<T extends object>(client: T, session: string | Session, options: WrapOptions): T
```

- `client`: The standard SDK instance (e.g., `octokit`).
- `session`: The active agent session ID or object.
- `options`: A `WrapOptions` object including the `connector` identifier (and optional `requireApproval` array).

**Returns:**  
A recursive Proxy of the original client (`T`), ensuring 100% type preservation.

## Advanced Methods

### `call()`

The manual executor. Use this when dealing with custom integrations, internal APIs, or manual back-outs not natively supported by the Rollback Registry.

**Signature:**  
```ts
call<T>(fn: Function, session: Session, undoConfig: UndoConfig, ...args: any[]): Promise<T>
```
