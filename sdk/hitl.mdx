## Approval Gates

Sometimes you want an agent to prepare an action, but hold it until a human reviews it. Using the `wrap()` method, you can easily enforce human approval gates on specific, highly-sensitive methods using the wrap options. 

Simply pass an array of methods into `requireApproval`:

```typescript
const safeStripe = agentrein.wrap(stripe, session, { 
  connector: 'stripe',
  requireApproval: ['subscriptions.cancel', 'refunds.create'] // Intercepts and pauses these calls
});

// This will halt execution until approved in the dashboard
await safeStripe.subscriptions.cancel({ id: 'sub_123' });
```

If the human rejects the action, the backend issues an exception to the SDK, bypassing the execution and automatically triggering a rollback of all previous session actions!
