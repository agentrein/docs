title: Built-in Connectors

## Overview

Built-in connectors are `ConnectorAction` objects that carry `apiName`, `operationType`, and rollback logic. When passed to `call()`, AgentRein reads these automatically — no need to set `actionName` or `operationType` manually.

For `UPDATE` and `DELETE` actions, the SDK calls `getState()` on the connector before execution to capture `beforeState` for snapshot-based rollback.

## Available Connectors

### Stripe
```ts
import { stripe } from 'agentrein/connectors';

stripe.invoices.create   // operationType: CREATE
stripe.invoices.del      // operationType: DELETE
stripe.customers.create  // operationType: CREATE
stripe.customers.del     // operationType: DELETE
stripe.refunds.create    // operationType: CREATE
```
