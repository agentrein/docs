## Overview

When you use `agentrein.wrap(client, session, { connector })`, AgentRein intercepts every method call and maps it to a built-in rollback strategy using the `connector` prefix and method path.

For example:
- `connector: 'github'` + method path `issues.create` → `apiName: github.issues.create` → rollback closes the issue
- `connector: 'slack'` + method path `chat.postMessage` → `apiName: slack.chat.postMessage` → rollback posts a correction message

## Supported SDKs

### GitHub — `connector: 'github'`

Install: `npm install @octokit/rest`

| Method Path | Operation | Rollback |
|---|---|---|
| `issues.create` | CREATE | Closes the issue |
| `issues.update` | UPDATE | Restores previous state |
| `repos.delete` | DELETE | No rollback available |

---

### Slack — `connector: 'slack'`

Install: `npm install @slack/web-api`

| Method Path | Operation | Rollback |
|---|---|---|
| `chat.postMessage` | CREATE | Posts a correction reply in thread |
| `chat.update` | UPDATE | Restores previous message |
| `chat.delete` | DELETE | No rollback available |

---

### Stripe — `connector: 'stripe'`

Install: `npm install stripe`

| Method Path | Operation | Rollback |
|---|---|---|
| `invoices.create` | CREATE | Deletes the invoice |
| `invoices.del` | DELETE | No rollback available |
| `customers.create` | CREATE | Deletes the customer |
| `customers.del` | DELETE | No rollback available |
| `refunds.create` | CREATE | No rollback available |

---

### HubSpot — `connector: 'hubspot'`

| Method Path | Operation | Rollback |
|---|---|---|

| `contacts.update` | UPDATE | Restores beforeState |
| `deals.create` | CREATE | Deletes the deal |
| `deals.update` | UPDATE | Restores beforeState |

---

### Gmail — `connector: 'gmail'`

| Method Path | Operation | Rollback |
|---|---|---|

### Google Sheets — `connector: 'gsheets'`

| `sheets.add` | CREATE | Deletes the sheet |

## Unsupported APIs

If your API is not listed above, use [`call()`](/sdk/call) with a custom [`undoConfig`](/sdk/undo-config):
```ts
const result = await agentrein.call(
  myApiFunction,
  session,
  createUndoConfig({
    type: 'http-delete',
    url: `https://api.example.com/resources/${resourceId}`,
    headers: { Authorization: `Bearer ${token}` },
  }),
  { actionName: 'myApi.resource.create' }
);
```
