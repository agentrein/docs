## Supported connectors

AgentRein includes built-in server-side rollback strategies for:

| Connector | Rollback strategy |
| --- | --- |
| Stripe | Voids created invoices, deletes created customers |
| Slack | Deletes posted messages |
| GitHub | Closes created issues |
| HubSpot | Deletes creates, restores `beforeState` for updates |
| Salesforce | Deletes creates, restores `beforeState` for updates |
| Notion | Moves pages to trash, deletes appended blocks |
| Gmail | Sends correction reply, untrashes, deletes drafts |
| Google Drive | Deletes files, restores metadata |
| Google Sheets | Clears ranges, deletes spreadsheets |

<Note>
  The SDK does **not** provide connector wrappers. Bring your own API client and pass it to `agentrein.call()`. The rollback strategies run automatically on the AgentRein server.
</Note>
