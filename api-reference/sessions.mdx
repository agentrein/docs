---
title: Sessions
description: Sessions represent a single agent run. All actions are scoped to a session.
---

## List Sessions
```http
GET /sessions
Authorization: Bearer <token>
```

Returns all sessions for the organization. Each session includes `_count.actions` (integer) instead of the full actions array.

---

## Create Session
```http
POST /sessions
Authorization: Bearer <token>
Content-Type: application/json

{
  "agentId": "billing-agent",
  "intent": "Cancel subscription and notify customer"
}
```

| Field | Type | Required |
|---|---|---|
| `agentId` | string | No — defaults to random UUID |
| `intent` | string | No |

**Response:** `201` — full Session object.

---

## Get Session
```http
GET /sessions/:id
Authorization: Bearer <token>
```

Returns full session including all actions and their snapshots.

---

## Complete Session
```http
PATCH /sessions/:id
Authorization: Bearer <token>
Content-Type: application/json

{ "status": "COMPLETED" }
```

Only allowed when session `status` is `ACTIVE`. Only `COMPLETED` is an accepted value.

---

## Session Statuses

| Status | Description |
|---|---|
| `ACTIVE` | Session is running |
| `COMPLETED` | Session finished successfully |
| `ROLLED_BACK` | Rollback was triggered and completed |
| `FAILED` | Session failed |

---

## GDPR Purge
```http
DELETE /sessions/:id/purge
Authorization: Bearer <token>
```

Redacts all `payload` and `response` fields in every action, and all `beforeState`/`afterState` in every snapshot. Also sets `intent` to null. Irreversible.

---

## Intent Drift

AgentRein uses GPT-4o-mini to evaluate every action against the session's original `intent`. A `driftScore` of `0.7` or above is flagged as drift.

### Get Drift Logs
```http
GET /sessions/:id/intent
Authorization: Bearer <token>
```

Returns one entry per action that has an intent log:

| Field | Description |
|---|---|
| `stepNumber` | Action order in the session |
| `apiName` | e.g. `stripe.invoices.create` |
| `originalIntent` | The session's `intent` string |
| `actualOutput` | The action's response |
| `driftDetected` | `true` if `driftScore >= 0.7` |
| `driftScore` | Float from `0.0` to `1.0` |
| `reason` | LLM explanation |
| `suggestion` | LLM suggestion |
| `createdAt` | ISO timestamp |

---

### Get Drift Summary
```http
GET /sessions/:id/intent/summary
Authorization: Bearer <token>
```

| Field | Description |
|---|---|
| `totalSteps` | Number of actions with intent logs |
| `driftedSteps` | Number of actions where `driftDetected` is `true` |
| `avgDriftScore` | Average drift score across all steps |
| `highestDrift` | `{ stepNumber, driftScore, reason }` or `null` |
