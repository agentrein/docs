## Get Metrics
```http
GET /metrics
Authorization: Bearer <token>
```

Returns aggregated stats across all sessions for the authenticated organization.

### Response

| Field | Type | Description |
|---|---|---|
| `totalActions` | number | Total actions logged across all sessions |
| `rollbacks` | number | Total actions with status `ROLLED_BACK` |
| `successRate` | number | Percentage of actions with status `SUCCESS` (0–100) |
| `avgLatency` | number | Average ms between session creation and its last action |
| `rollbackQueueDepth` | number | Current number of jobs in the BullMQ rollback queue |
| `estimatedClearTime` | string | Estimated time to clear the rollback queue |
| `connectorBreakdown` | array | Per-connector breakdown (see below) |

### Connector Breakdown

Each item in `connectorBreakdown`:

| Field | Type | Description |
|---|---|---|
| `connector` | string | Connector prefix e.g. `stripe`, `slack`, `github` |
| `totalActions` | number | Total actions for this connector |
| `successCount` | number | Actions with status `SUCCESS` |
| `rollbackCount` | number | Actions with status `ROLLED_BACK` |
| `failedCount` | number | Actions with status `FAILED` or `ROLLBACK_FAILED` |

### Example Response
```json
{
  "data": {
    "totalActions": 142,
    "rollbacks": 8,
    "successRate": 94.37,
    "avgLatency": 1203,
    "rollbackQueueDepth": 0,
    "estimatedClearTime": "Queue empty",
    "connectorBreakdown": [
      {
        "connector": "stripe",
        "totalActions": 60,
        "successCount": 57,
        "rollbackCount": 3,
        "failedCount": 0
      }
    ]
  },
  "error": null
}
```

<Info>
If Redis is unavailable, `rollbackQueueDepth` returns `0` and `estimatedClearTime` returns `"Queue empty"`. The endpoint does not fail.
</Info>
