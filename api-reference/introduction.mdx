The AgentRein REST API lets you manage sessions, trigger rollbacks, 
handle approvals, and fetch metrics programmatically.

## Base URL
https://api.agentrein.com

## Authentication

All endpoints require a Bearer JWT token obtained by exchanging 
your API key at `POST /auth/token`.
Authorization: Bearer <token>

See the [Authentication](/api-reference/authentication) page for details.

## Response format

All responses follow this envelope:
```json
{
  "data": { ... },
  "error": null,
  "timestamp": "2025-01-01T00:00:00.000Z"
}
```

Errors follow RFC 7807:
```json
{
  "data": null,
  "error": {
    "type": "https://agentrein.com/errors/unauthorized",
    "title": "Unauthorized",
    "status": 401,
    "detail": "Invalid API key"
  },
  "timestamp": "2025-01-01T00:00:00.000Z"
}
```
