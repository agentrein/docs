# AgentRein Documentation

This directory contains the [Mintlify](https://mintlify.com) documentation for AgentRein. The `.mdx` files here map directly to the live site.

## Live Docs
Read the published documentation at [agentrein.mintlify.app](https://agentrein.mintlify.app/).

## Running Locally

To preview changes to the documentation on your machine:

1. Open this directory in your terminal: `cd docs`
2. Run the Mintlify dev server: `npx mintlify dev`
3. Open `http://localhost:3000` in your browser.
